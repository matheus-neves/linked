# Linken
Save your favorite links in a simple way

## Dependencies

### CouchDB

Folow the [Setting up CouchDB](https://pouchdb.com/guides/setup-couchdb.html)

### NodeJS

[Download and Install](https://nodejs.org/en/)

## Installing
```
npm i -g webpack
npm i
```

## Running
```
'Development': npm run dev
'Production' : npm run build
```
